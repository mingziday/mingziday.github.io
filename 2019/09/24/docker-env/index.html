Docker环境配置

Fedora安装Docker

  1. 执行安装命令
    yum -y install docker

  1. 检查版本
    docker –version

  2. 启动 docker 进程。
    $ sudo systemctl start docker

  3. 开机时自动启动 Docker
    $ sudo systemctl enable docker
