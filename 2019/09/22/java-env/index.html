Java环境配置

JDK13安装

  1. 下载安装jdk-13_windows-x64_bin.exe
  2. 配置环境变量%JAVA_HOME%,maven需要这个环境变量

MAVEN安装

  1. 安装apache-maven-3.6.2-bin.zip
  2. 配置环境变量%MAVEN_HOME%
  3. 设置本地仓库地址：E:\repository
  4. 设置远程仓库阿里镜像： alimaven aliyun maven http://maven.aliyun.com/nexus/content/groups/public/ central

Eclipse安装（必须要有JDK）

远程链接Linux的JDK是不行的，但是调试的时候应该可以

  1. 安装eclipse-inst-win64.exe

  2. 安装Spring插件

  3. 配置Eclipse的MAVEN参数
    settings文件路径，D:\service\apache-maven-3.6.2\conf\settings.xml
